# Kabzhanov Homebrew Tap

Personal Homebrew tap for BizDNAi / AI Trust Index open-source tools.

## Install

```sh
brew tap kabzhanov/aisentinel
brew install aisentinel
```

Or in one command:

```sh
brew install kabzhanov/aisentinel/aisentinel
```

## Formulas

| Formula | Description | License |
| --- | --- | --- |
| [aisentinel](Formula/aisentinel.rb) | AI governance / ATI collector CLI & MCP server | Apache-2.0 |

## Updating

`brew update && brew upgrade aisentinel` to pull the latest stable tag.

For bleeding-edge:

```sh
brew install --HEAD kabzhanov/aisentinel/aisentinel
```
